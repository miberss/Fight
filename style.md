| Category               | Convention                                 |
|------------------------|--------------------------------------------|
| Constructor            | `a_*()`                                    |
| Lookup / Access        | `noun()`                                   |
| Mutator / Action       | `verb_*()`                                 |
| Module loader          | `load_*()`                                 |
| No get/create prefixes | forbidden                                  |
| Variables              | `snake_case`                               |
| Functions              | `snake_case`                               |
| Structs                | `PascalCase`                               |
| Struct fields          | `snake_case`                               |
| script file | `PascalCase` |
| library file | `snake_case` |
| Conditionals           | avoid nesting                              |
| Comments               | required and expressive                    |
| File structure         | notes, imports, experiments, structs, code |
